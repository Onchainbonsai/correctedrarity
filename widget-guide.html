Add the Script to Scriptable

Option A — File import (easiest):

1. Open the Files app on your iPhone
2. Find the downloaded BonsaiWidget.js

1. Open Scriptable
2. Tap + in the top right
3. Name it BonsaiWidget
4. Copy the code below and paste it in

4
Test the Script

Before adding it as a widget, test it works:

1. Open Scriptable
2. Tap BonsaiWidget
3. Tap the ▶ play button
4. You should see a widget preview with your rarest bonsai

Note: The first run takes ~10-15 seconds as it fetches data from OpenSea and the rankings file. Subsequent runs are faster due to caching.
5
Add Widget to Home Screen

1. Go to your iPhone Home Screen
2. Long press on an empty area until icons jiggle
3. Tap + in the top left corner
4. Search for "Scriptable"
5. Choose your preferred size: Small, Medium, or Large
6. Tap "Add Widget"
7. Tap the widget → select BonsaiWidget as the script
8. Tap Done

Pro tip: Choose Medium (2×1) for the best view — it shows the bonsai image plus name, rank and rarity info.
6
Pick Your Favourite Bonsai per Widget

Each widget can show any bonsai you own — your favourite, your rarest, or any specific one. Just set the Token ID as the widget parameter.

1. Long press the widget on your home screen → Edit Widget
2. In the Parameter field, type the Token ID of the bonsai you want
3. Tap outside to save — the widget updates immediately

Parameter: 1024  → shows Bonsai #1024
Parameter: 892   → shows Bonsai #892
Parameter: 3160  → shows Yuki (Named, #1)
Parameter: (empty) → shows your rarest bonsai automatically
How to find a Token ID: Go to the Rarity Checker → Holder Garden → enter your address → tap any bonsai → the Token ID is shown under the name.
The widget verifies ownership. If you enter a Token ID you don't own, the widget will show an error. You can only display bonsais in your wallet.
7
Auto-Update Settings

iOS widgets update automatically but you can influence the frequency:

— Scriptable refreshes widgets every 15-60 minutes depending on iOS
— Your bonsai's growth (Age trait) updates on-chain monthly
— New listings and prices update whenever the widget refreshes
— Tapping the widget opens the Rarity Checker website

Note: iOS controls widget refresh frequency automatically based on battery and usage patterns. You cannot force a specific interval, but Scriptable is generally refreshed every 15-30 minutes when your phone is in use.

Frequently asked questions

Why does the widget show "No bonsai found"?
Make sure the Ethereum address in the script (line 11) is correct and that wallet actually holds OnChain Bonsai NFTs. Double-check by entering the address in the Rarity Checker.
Why does the rank say "#9999 / 3200"?
The rankings.json file hasn't been generated yet, or that bonsai wasn't found in it. Visit the Global Ranking tab and generate it first.
Can I use this for someone else's wallet?
Yes! Just replace the ETH_ADDRESS in the script with any Ethereum address that holds bonsais. Great for keeping an eye on whale wallets.
Will the widget update when my bonsai grows older?
Yes. The Age trait updates on-chain over time, and the widget fetches fresh data on every refresh. You'll see the visual change reflected automatically.
Is my wallet address safe?
Your address is only used to read public data from OpenSea's API — it's a read-only operation. No private keys, no signing, no transactions. It's as safe as looking up an address on Etherscan.
The rarity rank is different from OpenSea. Why?
This widget uses corrected rarity — Age and Stage are excluded because they change over time and shouldn't define permanent rarity. OpenSea includes them, which inflates or deflates ranks unfairly. Our √(1/p) method gives a truer picture.
Can I use this on Android?
Scriptable is iOS only. On Android, you can use KWGT or Tasker with a similar JavaScript approach, but setup is more complex. The Live Garden page works on any device's browser.
